# Instagram-IUT-lens

## Intro

Projet scolaire réalisé au deuxième semestre de la première année.

## But

Récréer avec un framework développer par nos professeurs une version d'instagram focus sur notre Université.
Il y a possibilité de créer un compte, se connecter, publier des posts avec descriptions et tag.
Front développer en mobile first et le projet et finit celon ce que demander les cours
